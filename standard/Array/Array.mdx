## Array(method) last

const a = {}
const b = {}
const c = {}
const array = [a, b, c]

## Array(method) remove
``` typescript
const a = {}
const b = {}
const c = {}
const array = [a, b, c]
const isRemoved = array.remove(a)
// isRemoved - true
// array - [b, c]
```
